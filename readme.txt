# Dealership 

## Description 

A CRUD app OOP with implemented login system and hashed passwords - bcrypt for security, ability to list and buy vehicles from other users.

## Languages/Frameworks

HTML ,CSS, Python ,Bootstrap, Jinja, Flask, MYSQL
